## Clone esse projeto

```bash
> git clone https://github.com/Caussz/loja-DevWeb
```

## Instale as dependencias do site:

```bash
> cd loja-DevWeb
> npm install or npm i
```

### Para rodar
```bash
> npm run dev
> npm start
```
### Para Usar PM2
```bash
> npm i -g pm2 (para instalar)

> Para executar = pm2 start vite
> depois = pm2 monit
```
